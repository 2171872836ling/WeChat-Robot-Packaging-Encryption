WeChat-Robot-Packaging-Encryption
1. **No longer maintained** since **14 September 2024**; MIT license – use it however you like.  
2. A wrapper based on **pywx**, integrating **OpenAI** for automatic WeChat replies.  
   Required libraries: `pywin32`, `wxauto`, `openai`.  
3. The code is **triple-encrypted** to prevent key leakage; the original source is included.  
4. API keys have been removed—add your own key in `api_key.py` before running.
